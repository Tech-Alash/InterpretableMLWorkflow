Установка
====================

Изначально этот репозиторий использовался на Windows с Python 3.9 для воспроизведения опубликованных экспериментов.
Он не оформлен как wheel-пакет или переиспользуемая Python-библиотека, поэтому настройка здесь файловая и основана на запуске скриптов.

Что нужно заранее
========================================

Перед запуском скриптов проверь, что у тебя есть:

* Python, доступный в терминале;
* зависимости из ``requirements/requirements.txt``;
* исходный CSV FRED-MD, положенный в ``data/``;
* права на запись в ``results/``, потому что каждый этап workflow пишет туда файлы.

Рекомендуемая настройка окружения
==================================================

В `README` репозитория используется такой вариант:

.. code-block:: bash

   conda create -n workflow python=3.9.7
   activate workflow
   pip install -r requirements/requirements.txt

Если тебе удобнее ``venv``, эквивалент будет таким:

.. code-block:: bash

   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements/requirements.txt

Входные данные
==============================

Конфигурация по умолчанию ожидает snapshot FRED-MD, указанный в :mod:`helpers.config`:

* имя файла без расширения: ``2021-08``
* итоговый путь: ``data/2021-08.csv``

Репозиторий не скачивает датасет автоматически. Перед запуском :mod:`experiment`
нужно вручную положить CSV в каталог ``data/``.

Создание YAML-конфигурации экспериментов
====================================================================

Workflow читает настройки экспериментов из ``setup_files/main.yaml``.
Создать этот файл можно так:

.. code-block:: bash

   python setup_experiments.py

Скрипт записывает YAML на основе значений по умолчанию, заданных в :func:`setup_experiments.build_experiment_setup`.

Локальная сборка документации
====================================================

Сайт документации можно собрать отдельно от окружения с экспериментами:

.. code-block:: bash

   pip install -r docs/requirements.txt
   sphinx-build -b html docs docs/_build/html

Главная страница собранного сайта будет здесь:

.. code-block:: text

   docs/_build/html/index.html
