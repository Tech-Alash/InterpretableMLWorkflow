Введение
========

Этот репозиторий является кодовой реализацией исследования
``An interpretable machine learning workflow with an application to economic forecasting``.
В статье авторы предлагают не просто обучить более точную модель, а выстроить
цельный процесс использования машинного обучения в среде, где важны:

* качество прогноза;
* объяснимость результата;
* возможность обсуждать выводы с аналитиками, менеджментом и конечными стейкхолдерами;
* статистическая проверка того, что интерпретации действительно устойчивы.

Почему это исследование важно
-----------------------------

Во многих прикладных задачах, особенно в экономике и policy-анализе, недостаточно
сказать, что модель дала хороший прогноз. Нужно ещё ответить на вопросы:

* почему модель пришла именно к такому выводу;
* какие переменные действительно важны;
* можно ли доверять найденным зависимостям;
* как объяснить результат не только технической команде, но и людям, принимающим решения.

Именно здесь возникает основной конфликт. С одной стороны, machine learning модели
часто точнее простых линейных подходов и умеют улавливать нелинейности и взаимодействия.
С другой стороны, они нередко воспринимаются как black box, из-за чего их сложно
использовать в институциональной среде, где интерпретация результата так же важна,
как и сам прогноз.

Главная идея статьи
-------------------

Статья предлагает трёхшаговый workflow, который помогает примирить точность и интерпретируемость:

1. сначала сравнить модели по качеству прогноза;
2. затем разобрать, какие признаки и каким образом влияют на прогноз;
3. после этого провести статистический вывод на основе Shapley-декомпозиций.

Такой подход позволяет использовать machine learning не как изолированный black box,
а как часть аналитического процесса, где результаты можно проверять, обсуждать
и использовать в объяснимой форме.

Кейс исследования
-----------------

В статье workflow демонстрируется на задаче прогнозирования изменений уровня
безработицы в США на горизонте 12 месяцев вперёд. Для этого используется набор данных
FRED-MD с ежемесячными макроэкономическими показателями.

В базовой постановке:

* целевая переменная - изменение безработицы;
* горизонт прогноза - 12 месяцев;
* данные - макроэкономические ряды FRED-MD;
* baseline-сравнение - простые линейные и авторегрессионные модели против более гибких ML-подходов;
* интерпретация - через Shapley values и permutation importance;
* статистическая проверка - через Shapley regression.

Ключевые выводы статьи
----------------------

Основные выводы исследования, которые полезно держать в голове при чтении кода:

* более гибкие machine learning модели могут превосходить линейные подходы по точности;
* природа этого преимущества часто связана с нелинейными и меняющимися во времени зависимостями;
* Shapley-декомпозиции позволяют увидеть содержательные экономические паттерны, которые модель действительно выучила;
* статистический вывод поверх Shapley values делает интерпретацию ближе к привычной эконометрической практике;
* лучший баланс между интерпретируемостью и качеством часто достигается не на максимально большом числе признаков, а на компактном наборе переменных, выбранных экспертно.

Что даёт репозиторий
--------------------

Этот репозиторий нужен не для абстрактной демонстрации отдельного алгоритма,
а для воспроизведения полного workflow из статьи:

* подготовки и трансформации данных;
* запуска матрицы forecasting-экспериментов;
* оценки качества моделей;
* расчёта feature importance;
* агрегации результатов;
* последующего анализа и интерпретации.

Поэтому документацию для этого проекта удобнее читать не как API manual,
а как объяснение исследовательского конвейера, где код, результаты и главы статьи
связаны между собой.
